# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | ✅ Active development |
| < 1.0   | ❌ Pre-release      |

## Reporting a Vulnerability

We take the security of GitHub Discovery seriously. If you believe you've found a security vulnerability, please report it by email to the maintainers.

**Do not open a public GitHub issue for security vulnerabilities.**

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You should receive a response within 48 hours. We appreciate your help in keeping the project secure.
